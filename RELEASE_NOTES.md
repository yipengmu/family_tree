# Release Note v0.1

更新范围：2026-07-13 至 2026-07-16

本阶段围绕“更容易开始建谱、更方便持续记录、更安心查看家谱”完成了一轮核心体验升级。

## 核心更新

- **首次建谱向导**：新用户可以从本人开始，按步骤补充父母和祖辈；知道多少记录多少，不必先准备完整族谱，保存后即可看到第一版家谱。
- **纸谱照片合并建谱**：空白家谱支持一次上传 1–10 张纸谱照片，系统合并识别跨页文字和世系关系，先生成预览，确认后才写入家谱，减少手工录入成本。
- **逐代续录家谱**：沿“本人 → 父亲 → 祖父 → 曾祖父”的主线逐代补录，保存一代后立即回到家谱查看新增关系，降低继续建谱的理解和操作门槛。
- **人物生平记录**：可以进入具体人物，记录一段文字、录音或照片故事，并将原始材料保留为家庭档案依据；AI 仅提供待确认草稿，用户确认后才发布。
- **移动端使用体验**：手机端默认进入 H5，续家谱、人物资料和家谱漫游界面更适合单手操作；家谱节点与人物信息更紧凑，折叠屏展开时也能更充分利用空间。
- **人物资料与隐私控制**：支持按人物增量保存资料，减少修改时的等待和覆盖风险；在世人物的出生日期、住址、联系方式等敏感字段会按家谱隐私设置展示。
- **官网与示范谱体验**：官网、登录和家谱应用入口重新梳理；示范家谱的布局、代际关系和人物称谓展示更清晰，访客更容易理解产品并开始自己的家谱。
- **密码找回与重置**：忘记密码后可通过邮箱验证码设置新密码，恢复账号访问。
