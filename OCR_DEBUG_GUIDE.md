## 🚀 使用方法

### 1. **启用调试模式**

确保 `.env` 文件中设置：
```env
REACT_APP_DEBUG=true
REACT_APP_ENV=development
```

### 2. **使用调试功能**

1. **上传图片**到OSS
2. **点击"显示调试"**按钮
3. **使用"生成测试数据"**快速测试表格填充
4. **点击"开始识别"**进行OCR识别
5. **查看调试面板**的实时日志
6. **监控数据状态**确认数据正确传递

### 3. **调试步骤**

#### 步骤1：验证数据生成

## 🔍 常见问题排查

### 问题1：OCR识别返回空数据
**解决方案：**
- 检查图片URL是否有效
- 验证OCR服务配置
- 使用"生成测试数据"功能验证表格

### 问题2：数据生成但表格不显示
**解决方案：**
- 检查数据格式是否正确
- 确认必需字段存在
- 使用强制刷新功能

### 问题3：数据字段不匹配
**解决方案：**
- 查看数据状态监控
- 检查字段映射
- 验证列定义
