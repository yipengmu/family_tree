### 🚀 使用流程

#### 1. 开发环境启动
```bash
# 复制环境变量配置
cp .env.example .env

# 启动应用
npm start
```

#### 2. 创作流程
1. **选择租户**: 在右上角选择或创建家谱
2. **上传图片**: 选择最多10张家谱图片上传
3. **OCR识别**: 点击识别按钮，自动解析图片内容
4. **数据校对**: 在表格中校对和编辑识别结果
5. **保存数据**: 保存到当前家谱或下载JSON文件

#### 3. 多租户管理
- **创建家谱**: 点击"创建新家谱"，填写信息
- **切换家谱**: 在下拉菜单中选择不同家谱
- **删除家谱**: 点击删除图标（默认家谱不可删除）
