# 测试文件目录

本目录包含项目的所有测试代码和调试文件，按类型分类存放：

## 目录结构

- `unit/` - 单元测试
- `integration/` - 集成测试
- `e2e/` - 端到端测试
- `performance/` - 性能测试
- `debug/` - 调试相关脚本

## 测试文件说明

### 集成测试 (integration/)
包含各种功能集成测试脚本，如：
- OCR数据测试
- OSS连接测试
- 代理测试
- 网络诊断测试
- 超时设置测试
- 后端OCR测试等

### 调试脚本 (debug/)
包含调试相关的脚本，用于问题排查和调试。

### 端到端测试 (e2e/)
包含端到端测试文件，如API测试页面。
