## 🔧 故障排除

### 1. 后端服务启动失败
```bash
# 检查端口占用
lsof -i :3003
# 杀死占用进程
kill -9 <PID>
```

### 2. 前端无法连接后端
- 确认后端服务已启动
- 检查端口3003是否可访问
- 查看防火墙设置

### 3. 千问API超时
- 检查网络连接
- 尝试使用VPN
- 增加超时时间（在server/app.js中修改）

### 4. API Key问题
- 检查`.env`文件中的`REACT_APP_QWEN_API_KEY`
- 确认API Key有效且有权限
